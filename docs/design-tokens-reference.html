Preattentive Attributes

  • COLOR: Role/stage identification (see above)
  • SIZE: Importance (1.8√ó banner, 1.5√ó top event, 1.0√ó standard)
  • SHAPE: Circle = top event ONLY, rectangles for all others
  • OPACITY: 1.0 = active, 0.3 = dimmed
  • GLOW: Role-colored box-shadow for active emphasis
  • MOTION: 1-2 pulse cycles on activation, then static
  • POSITION: Left = prevention, Center = top event, Right = mitigation
  • ACCENT: Left border bar for threats (amber)

Design Principle: One pattern = one meaning. Each visual attribute maps to exactly one semantic category. No reuse across different roles. Maximum clarity at a glance.
